Programando a los cuarenta

¿Hay una edad para poder aprender algo nuevo como un lenguaje?. Un idioma nuevo, inglés, chino, portugués o catalán. Aprender programación es lo mismo que entraras a un curso de idiomas. Hay diferentes lenguajes que puedes aprender y muchos de ellos se potencian al trabajar juntos, por ejemplo, como el html5. HTML, JAVA y CSS son 3 lenguajes que usan sus propias reglas para escribir código y de esa manera, comunicarse con los computadores. Sus alcances son muchos: APPS, páginas webs, juegos, y softwares.

"Programando a los cuarenta" es un blog para desafiar la idea de que el código no se puede aprender a cualquier edad.
Todo parte entendiendo la lógica del pensamiento computacional y termina cuando cuando se cierre la gran etiqueta de la vida.

Este Blog es parte del proyecto "Irremplazables": una docuserie dedicada ayudar a las personas a sobrellevar la 4ta y 5ta revolución industrial.
Mi nombre es Viviana Garrido y soy la creadora de la serie. Como guionista y directora, no podía hablar de código sin haberlo experimentado.
